# Illustrations -- Source Files
All of the source files for the illustrations used in Clean World are contained in the folder "IllustrationSourceFiles".

**Please note that if you're just looking for the exported ".png" versions of the illustrations, you can get them under the "Assets/Sprites" folder in the Unity project.**

I apologize in advance for the messiness of the illustration source files. I never really intended for them to be viewed, but I decided that it would likely be best to back them up here.

<br></br>

## Opening Illustration Source Files
The ".svg" files contained in the "IllustrationSourceFiles" folder must be opened in a free piece of software called "Inkscape". 
Opening them with another tool will likely prevent you from editing them and cause them to not render correctly. Also, most of the illustrations flow off of the designated canvas area, so you may need to zoom out to see everything.
